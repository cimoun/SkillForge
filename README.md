# SkillForge

Планировщик обучения с визуализацией прогресса навыков.

## Идея

Человек описывает набор навыков, которыми хочет владеть, привязывает к ним учебные активности и отслеживает прогресс через радар-диаграмму.

## Возможности

- **Карта навыков** — добавляй навыки с целевым уровнем и приоритетом
- **Активности** — курсы, книги, проекты, практика
- **Связи** — каждая активность качает определённые навыки с разным весом
- **Логирование времени** — трекай часы, уровни вычисляются автоматически
- **Радар** — визуализация текущего состояния vs цели
- **План недели** — рекомендации, что качать в первую очередь

## Формула уровня

```
weighted_hours = Σ (activity_hours × skill_weight)

0h      → 0 (не начинал)
1-10h   → 1 (знакомство)
11-40h  → 2 (основы)
41-100h → 3 (практик)
101-250h → 4 (уверенный)
251h+   → 5 (эксперт)
```
